# Task 3 – Dashboard Design
Hi! This is my submission for Task 3 of the Data Analyst Internship – focused on Design an interactive dashboard for business stakeholders.. The dataset I worked with is called Superstore.csv`, which contains Total Sales,Total Profit,Total Orders,Region,Segment,and Category. Here’s a summary of what I did:
  
** Visuals Created:**
- KPI Cards: Total Sales, Total Profit, Total Orders
- Monthly Sales Trend (Line Chart)
- Top Sub-Categories by Sales (Bar Chart)
- Segment-wise Profit Share (Donut Chart)
- Sales by Region (Bar Chart)
- Sales & Profit by State (Map View)

** Tools Used:**
- Tableau Public
- MS Excel (data preview)

** Key Insights:**
- **Corporate** segment contributed the highest profit.
- **California** led in both sales and profit.
- **November** showed the highest sales volume.
